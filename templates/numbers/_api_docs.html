Bitcoin Numbers

A provably fair "numbers" game, powered by Bitcoin block data.

Set a date/time, odds, and at that time the latest bitcoin block hash will be fetched and a winning number calculated.

How it works:
On the closing date, the game takes the last block hash and converts it to a number between 0 and (whatever you set the odds to).
The higher the Odds, the more difficult it is to win/the more users win (-haircut).
Max Odds is 10,000,000-1

Risks:
This game relies on block data from mempool.space, which means you're trusting them not to manipulate it.
In theory, this game could skewer mining incentives, if the prize were so big its within miners interest to mine a block with certain hash ending.
